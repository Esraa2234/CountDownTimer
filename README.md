# Countdown Timer 

A simple countdown timer that counts down to a specific date and time. Updates every second and displays days, hours, minutes, and seconds.

---

##  Project Overview
This project is designed to show a real-time countdown to a target date. It demonstrates how to manipulate the DOM dynamically using JavaScript and update elements in real time.

---

##  Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## Features
- Displays countdown in days, hours, minutes, and seconds
- Updates every second automatically
- Handles countdown completion (shows "00" when the date is reached)
- Clean and simple UI


---

##  Concepts Used
- DOM manipulation (`getElementById`, `innerHTML`)
- Time calculations using `Date` objects
- `setInterval` for periodic updates
- Conditional statements for countdown completion
- Modular and reusable JS code
